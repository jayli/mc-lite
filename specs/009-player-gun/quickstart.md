# Quickstart: 009-player-gun

## Test Scenarios

### 1. 模型加载测试
- 启动游戏。
- 检查控制台是否有加载 `gun.gltf` 失败的错误。
- 确认 `Engine.gunModel` 在加载后不为 `null`。

### 2. 状态切换测试
- 进入游戏后，按下 `R` 键。
- **预期**: 右下角出现枪支模型，原本的手臂（如果有显示动作）消失。
- 再次按下 `R` 键。
- **预期**: 枪支模型消失。

### 3. 第一人称视角测试
- 在持枪状态下，左右转动鼠标。
- **预期**: 枪支稳固地保持在屏幕右下角。
- 向前、向后移动。
- **预期**: 枪支不发生抖动，始终位于视野内。

### 4. 视角限制测试
- 向上或向下看到极限位置。
- **预期**: 枪支模型不穿插进相机内部（裁剪），且不遮挡准星。
